# Information Classification for Confluence

A Forge app that adds information classification levels to Confluence Cloud pages.

## Features

- Classify pages with configurable levels (Public, Internal, Confidential, Secret)
- Colored badge in the page byline showing the current level
- Restriction warnings when page access doesn't match classification requirements
- Recursive classification for page trees
- Per-space level restrictions and defaults
- Audit trail with charts and history
- Multi-language support (English, German, French, Japanese)
